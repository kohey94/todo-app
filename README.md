# 技術
- MongoDB
- Express
- React
- Node.js
- Jest
  
## Available Scripts
### サーバー起動
#### `node backend/server.js`

### フロント起動 
#### `npm start`  
Open [http://localhost:3000](http://localhost:3000) to view it in your browser.

### テスト
#### `cd backend`
#### `npm test`
